# Interview Assistant — 面试助手

一个帮助面试复习的 Markdown 问答工具，支持答案隐藏/显示切换和错题集管理。

## 功能

- 📝 支持 Markdown 格式的问答笔记（`# Q:` ~ `###### Q:` 任意级别标题）
- 👁️ 复习模式默认隐藏答案，点击显示
- 📌 错题集：收藏不熟悉的知识点，便于集中复习，掌握后可删除
- 🌓 亮色/暗色主题
- ⌨️ 键盘快捷键（`Shift+A` 切换全部，`Shift+E` 编辑，`Shift+R` 复习，`Shift+W` 错题集）
- 🖨️ 打印友好（自测模式）
- 🧩 浏览器插件（支持 file:// 和 GitHub 页面）

## 项目结构

```
├── core/           # 核心 Markdown Q&A 解析器
├── extension/      # Chrome/Edge 浏览器插件
├── web/            # 单文件 Web App
└── README.md
```

## 使用方式

### 🌐 在线体验
直接访问：**https://jobn123.github.io/interview-assitant-markdown/**

### Web App
直接用浏览器打开 `web/index.html`，粘贴或拖拽 `.md` 文件即可。

### 键盘快捷键

| 快捷键 | 功能 |
|--------|------|
| `Shift + A` | 显示 / 隐藏全部答案（复习模式或错题集模式） |
| `Shift + E` | 切换到编辑模式 |
| `Shift + R` | 切换到复习模式 |
| `Shift + W` | 切换到错题集模式 |

### 📌 错题集

将不容易掌握的知识点加入错题集，便于后续集中复习：

- **添加**：在复习模式下点击卡片右侧的 📌 书签按钮
- **手动添加**：在错题集页面点击「➕ 手动添加」，输入知识点和答案
- **复习**：切换到「📌 错题集」标签，与复习模式相同的卡片交互体验
- **删除**：掌握后点击 ❌ 按钮移除，或使用「🗑️ 清空」一键清空
- 数据自动保存到浏览器，刷新页面不丢失

### 浏览器插件
1. Chrome 打开 `chrome://extensions`
2. 开启"开发者模式"
3. 点击"加载已解压的扩展程序"，选择 `extension/` 目录
4. 打开本地 `.md` 文件或 GitHub 上的 Markdown 文件
