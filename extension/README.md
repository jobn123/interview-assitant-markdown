# Interview Assistant — 浏览器插件

## 安装步骤

### Chrome / Edge

1. 打开扩展管理页面：
   - Chrome: `chrome://extensions`
   - Edge: `edge://extensions`

2. 开启右上角的 **「开发者模式」**

3. 点击 **「加载已解压的扩展程序」**，选择 `extension/` 目录

4. **重要**：在扩展卡片中找到 Interview Assistant，点击 **「详细信息」**，
   开启 **「允许访问文件网址」**（这样才能在本地 .md 文件上使用）

### 使用方式

- **本地 .md 文件**：直接在浏览器中打开 .md 文件（`file://` 协议）
- **GitHub**：浏览任意仓库的 `.md` 文件 blob 页面
- **GitLab**：浏览任意仓库的 `.md` 文件 blob 页面
- **Raw 文件**：访问 raw.githubusercontent.com 上的 .md 文件

插件会自动检测页面中的 Q&A 内容，并：
- 隐藏所有答案
- 在每个问题旁添加 + 按钮
- 右上角显示控制工具栏

## 支持的格式

- `# Q:` `## Q:` `### Q:` ... `###### Q:` — 问题标题（任意级别）
- `# A:` `## A:` `### A:` ... — 显式答案标题（可选）
- 隐式答案：Q 标题之后、下一个标题之前的内容
