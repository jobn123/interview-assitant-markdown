# 面试助手（Interview Assistant）— 用 Markdown 高效刷题，面试复习从未如此简单

## 这是什么？

**面试助手**是一个基于 Markdown 的轻量级问答复习工具。你只需要用 Markdown 写下面试知识点和答案，它就能自动解析成**交互式 Q&A 卡片**——默认隐藏答案，点击才显示，模拟真实面试的自问自答场景。

> 🌐 **在线体验**：https://jobn123.github.io/interview-assitant-markdown/

## 它解决了什么问题？

如果你在准备技术面试，大概率会用 Markdown 整理复习笔记——这是开发者的本能。但**看笔记 ≠ 记住笔记**：

- ❌ 打开 .md 文件，问题和答案一目了然，**眼睛扫过去就以为"我会了"**，实际上根本没记住
- ❌ 用 Notion/语雀之类的工具，格式臃肿、加载慢，没有"遮住答案自测"的体验
- ❌ 想针对薄弱知识点集中复习，只能手动翻找，**没有错题集机制**
- ❌ 在 GitHub 上看到好的面试题笔记，只能看不能交互

**面试助手就是为解决这些痛点而生的**。它把静态的 Markdown 笔记变成一套完整的复习系统：

- 👁️ **答案默认隐藏**，逼迫你先思考再确认，告别 "眼睛会了手不会"
- 📌 **内置错题集**，不熟的知识点一键收藏，集中攻克薄弱环节
- ⚡ **零依赖、零构建**，一个 HTML 文件直接用浏览器打开，无需安装任何东西
- 🧩 **浏览器插件**，在 GitHub 上浏览 Markdown 面试题时，直接注入交互卡片

## 核心特点

### 1. 智能 Markdown 解析，怎么写都能识别

你不需要学习特殊语法。以下写法**全都能被正确解析**：

```markdown
## Q: 什么是闭包？
### 问题：什么是原型链？
### Q1: HTTP 和 HTTPS 的区别？
### 1. 事件循环
```

支持中文/英文标记、全角/半角冒号、编号格式、甚至纯文本标题自动推断——**你按自己的习惯写就行**，解析器自动归一化。

### 2. 错题集——面试复习的核心闭环

这是面试助手区别于普通 Markdown 阅读器的关键功能：

- 复习时遇到不熟的知识点，点击 📌 **一键收藏**到错题集
- 切换到错题集标签页，**只看薄弱环节**
- 掌握后点击 ❌ 移除，或一键清空重新来过
- 数据自动保存到浏览器 `localStorage`，**刷新不丢失**

### 3. 零依赖，极致轻量

整个 Web 应用是**一个 HTML 文件**，包含所有 HTML、CSS、JavaScript。没有 npm、没有 node_modules、没有构建步骤。下载到本地，双击打开就能用——断网也能用。

### 4. 浏览器插件，随处可用的复习工具

安装了 Chrome/Edge 插件后：

- 打开**本地 .md 文件** → 自动变成交互式复习页面
- 浏览 **GitHub 上的 Markdown 面试题仓库** → 直接注入显示/隐藏按钮
- 其他任意包含 Markdown 的页面 → 一键弹出复习窗口

### 5. 键盘快捷键 + 亮暗主题

| 快捷键 | 功能 |
|--------|------|
| `Shift + R` | 复习模式 |
| `Shift + W` | 错题集 |
| `Shift + E` | 编辑模式 |
| `Shift + A` | 一键切换全部答案 |

亮色/暗色主题自动跟随系统或手动切换，深夜复习不刺眼。

### 6. 打印友好，线下自测

复习模式下直接打印，**生成的纸张版就是一张自测卷子**——问题可见，答案隐藏，适合脱离屏幕背诵。

## 适合谁用？

- 🎯 **准备技术面试的开发者** — 前端、后端、算法，任何能用 Markdown 记录的知识点
- 📚 **构建个人知识库的人** — 把你的学习笔记变成可自测的题库
- 👥 **面试官 / 培训师** — 用 Markdown 编写面试题库，分享给候选人自测

## 怎么开始？

1. **在线使用**：直接访问 [GitHub Pages](https://jobn123.github.io/interview-assitant-markdown/)，粘贴 Markdown 内容即可
2. **本地使用**：下载 `web/index.html`，双击打开
3. **浏览器插件**：Chrome 加载 `extension/` 目录，在所有 .md 页面上生效
4. **示例笔记**：项目自带 `sample-notes.md`，覆盖了所有支持的格式
